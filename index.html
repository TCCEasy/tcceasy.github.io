Trabalho científico ou de revisão?

   
  Django   Jinja   Python   ChartJS   HTML   CSS   Bootstrap   JavaScript   jQuery   SQLite

É importante também definiar com antecedência se você fará uma pesquisa de campo ou um trabalho de revisão. Qual a diferença de um para o outro? Bem simples! O trabalho científico envolve uma pesquisa seja através de experimentos, seja através de entrevista/questionário, e também de pesquisa da literatura (livros, artigos, teses) para comparar trabalhos semelhantes ao seu e fortalecer suas discussões. Já o trabalho de revisão apenas foca a pesquisa de trabalhos escritos já feitos, de preferência mais recentes e publicados em revistas de boa qualidade.

Etapas na criação de um TCC:

   
  Django   Python   HTML   CSS   Bootstrap

Em suma, nós temos: introdução, metodologia, resultados, discussão, conclusão e referências bibliográficas. Mas também aparecem com frequência: objetivos, justificativa, referencial teórico, cronograma. Perceba que são várias etapas, é um processo trabalhoso, porém é simples se você se dedicar e não deixar tudo pra última hora. A única parte mais complexa de um TCC talvez sejam as discussões, pois nesta etapa você vai comparar diferentes autores e, com base no que você pesquisou, sugerir ou propor um direcionamento para a comunidade científica acerca de um assunto específico que você esteja trtando em seu trabalho.
